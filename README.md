# Home Advantage in Ekstraklasa 1989-2023

## Project Overview
Analysis of home advantage in Polish football's top division — Ekstraklasa.
The project explores whether playing at home significantly impacts match results
and how this advantage has changed over 30 years.

## Questions Explored
- Does home advantage differentiate champions from the rest of the table?
- Has home advantage declined over the last 30 years in Ekstraklasa?

## Dataset
- Source: [Kaggle — Ekstraklasa 1989-2023][(https://www.kaggle.com/)](https://www.kaggle.com/datasets/marcinkamil84/ekstraklasa-1989-2023)
- 9105 matches across 32 seasons

## Tools & Libraries
- Python
- pandas
- matplotlib

## Key Findings
- Home advantage does not significantly differentiate champions from the rest of the table — all teams perform better at home regardless of final position.
- Home advantage has clearly declined over 30 years — in the 1990s the difference between home and away win % was around 25-30 points, dropping to 10-15 points in recent seasons.
